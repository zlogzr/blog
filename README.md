### 多人博客管理系统
技术栈：nodejs+express+art-template模版引擎+mongodb数据库
#### 启动
1. node app.js
2. 打开浏览器访问地址localhost
3. 点击登录 管理员登录进入后台管理 管理员账号：zlog@zlog.com 密码：123456
普通用户登录进入首页能进行评论 普通用户账号：wangwu@zlog.com 密码：123456
